OVERVIEW
========
Professional wireframes for the NGIN Alliance Playbook microsite demonstrating solid UX flow, 
information architecture, and page flow. All content is real text—no lorem ipsum or placeholder lines.

7 WIREFRAMES INCLUDED
=====================

1. HOME PAGE (playbook.newgrowth.org)
   - Hero section with CTA buttons
   - "Building Civic Infrastructure That Lasts" section with two-column layout
   - "Start Where You Are" pathway cards (3-column)
   - "What's Inside" content cards (2x2 grid)
   - Stats bar with key metrics
   - Sticky sidebar navigation

2. ABOUT THE ALLIANCE (playbook.newgrowth.org/about)
   - Program origins and mission
   - Why the alliance was created
   - Four-part approach cards (Customized Training, Facilitation, Technical Assistance, Peer Learning)
   - Two program phases
   - Three program objectives

3. FRAMEWORK & INSIGHTS (playbook.newgrowth.org/framework)
   - Four building blocks with color-coded sections:
     * Trust (teal) - 5-step sequence + field example
     * Shared Narrative (amber) - 3 jobs of narrative + example
     * Facilitation & Project Management (gray) - 5 capabilities + example
     * Structure & Shared Practice (green) - 5 core elements
   - "Practice Makes Durable" diagram section
   - 60-Day Quick Start in full-width navy box

4. RESOURCE BANK (playbook.newgrowth.org/resources)
   - Search bar with placeholder
   - Filterable tags (All Tools, Trust, Narrative, Facilitation, Structure)
   - 15 tool cards in 3-column grid with descriptions and tags
   - Each tool includes building block association

5. CITIES INDEX (playbook.newgrowth.org/cities)
   - 9 city case study cards in 3-column grid
   - Each card shows: city image placeholder, theme, partner organizations, description
   - Cities: Bangor ME, Omaha NE, Honolulu HI, Detroit MI, Sarasota FL, 
     Cleveland OH, Shreveport LA, Sacramento CA, New Orleans LA

6. CASE STUDY TEMPLATE (Bangor, ME example)
   - Full-width layout (no sidebar)
   - Quick Info strip (4-column: Organizations, Building Blocks, Related Tools, Download)
   - Sections: Organizations Involved, Community Context, Type of Project, Alliance Interventions,
     Challenges Faced, Successes and Outcomes, Lessons Learned
   - Do's and Don'ts (2-column with checkmarks)
   - Related Case Studies (3-card carousel at bottom)

7. TOOL PAGE TEMPLATE (Positionality Statement example)
   - Full-width layout (no sidebar)
   - Quick Info strip
   - Overview section with real content
   - "When to Use This Tool" section
   - 5-step "How to Use" with numbered circles
   - 5 accordion sections (expandable) for template sections
   - "Tips for Facilitators" with callout boxes
   - Related Tools section with 3 cards

DESIGN SPECIFICATIONS
=====================
Color Palette:
  - Navy (Primary): #1B3A6B
  - Teal (Accent 1): #2E7D6B
  - Amber (Accent 2): #D97706
  - Light Gray (BG): #F3F4F6
  - Dark Gray (Text): #111827

Layout:
  - Left sidebar: 230px fixed width (sticky), navy background
  - Main content: max 900px width, centered
  - Responsive: Sidebar collapses to full width on tablets/mobile
  - Browser frame: Chrome-style with dots and URL bar

Typography:
  - H1: 36px, bold
  - H2: 28px, bold
  - H3: 20px, semi-bold
  - H4: 16px, semi-bold
  - Body: 15px, line-height 1.7
  - System font stack
